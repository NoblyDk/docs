## What is Nobly Insight?

Nobly Insight is an AI-powered document intelligence and collaboration platform. It brings together intelligent search, document management, workflow automation, and redaction tools in a single interface — helping you find, manage, and act on documents faster.

## Core features

<Columns cols={2}>
  <Card title="Search" icon="magnifying-glass">
    Find documents using traditional keyword search, structured filters, or AI-powered semantic search that understands the meaning behind your queries.
  </Card>
  <Card title="Upload" icon="cloud-arrow-up">
    Upload documents with keyword and metadata assignment, supporting multiple file types and Office integration.
  </Card>
  <Card title="Caseflow" icon="table-columns" href="/caseflow/introduction">
    Browse applications and classes, apply filters, create and manage workflow objects, and track work across your organization.
  </Card>
  <Card title="Redaction" icon="eraser">
    Redact and annotate documents with built-in markup tools, OCR support, and an AI-powered redaction assistant.
  </Card>
</Columns>
