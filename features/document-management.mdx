## Overview

The heart of Nobly Insight: capture, store, find, and work with documents at enterprise scale. Every document is classified, secured, audited, and reachable from a modern browser interface — no client install required.

## Storage and retrieval

- Centralised, secure storage for any file type — Office documents, PDFs, images, e-mails, scanned paper, and more
- Document **metadata** (keywords) for fast retrieval and classification
- Full **revision history** for every document — view, restore, or compare older versions
- Granular permission model so users only see the documents they are entitled to

## Capture and ingestion

| Capability | What it does |
|---|---|
| **Drag-and-drop upload** | Add files directly from the browser |
| **Chunked upload** | Reliably upload very large files without practical size limits |
| **Bulk and batch capture** | Move high-volume scanning workflows into the platform |
| **Office add-ins** | Save e-mails and Office documents straight from Outlook, Word, Excel, and PowerPoint without leaving the application |
| **AI-assisted upload** | Suggests a document type and pre-fills metadata from the file content during upload |

See [How we use AI](/ai/introduction) for how the AI-assisted upload feature handles your data.

## Editing and viewing

- Built-in **viewer** for PDFs, images, and Office files — no client install required
- **Office Online editing**: open and edit Word, Excel, and PowerPoint documents directly in the browser, with auto-save back to the platform
- **Native Office editing**: round-trip editing with the desktop versions of Office
- **Document composition**: generate new documents from templates with merged data from cases and forms
- **Notes and collaboration**: threaded notes per document with @mentions, reminders, and follow — see [Notes & collaboration](/features/notes-and-collaboration)

## Redaction

- Redact sensitive information (names, account numbers, personal data) before sharing
- **AI-assisted redaction**: automatic detection of personal data, identifiers, and other sensitive content for one-click human review and redaction
- **Permanent or reviewable** redaction modes

A human reviewer always confirms suggested redactions before they are applied.

## Document lifecycle

- Configurable **document types** with their own metadata, security, and retention rules
- **Lifecycle events** (create, update, view, delete) are emitted on a real-time event stream so other systems can react
- **Document retention** policies for automatic, audit-friendly disposal

## Real-time collaboration and notifications

- **Real-time updates** via push notifications: when a document is updated, every user viewing it sees the change immediately
- **Collaborative editing** of Office documents in the browser with co-authoring
- Live workflow inbox with new-task alerts (see [Case management](/features/case-management))
